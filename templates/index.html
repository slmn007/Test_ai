Klasifikasi Penggunaan Masker pada Wajah
Aplikasi ini terdiri dari dua tahapan utama, yaitu deteksi lokasi wajah dan klasifikasi penggunaan masker pada wajah yang telah terdeteksi.

Tahapan deteksi lokasi wajah menggunakan modul 'face recognition' yang didapatkan dari https://pypi.org/project/face-recognition/. Pada modul ini diperoleh lokasi wajah pada suatu citra, yaitu koordinat kotak deteksi (top, left, bottom, right).

Tahapan klasifikasi akan mengklasifikasikan wajah yang terdeteksi dari tahapan pertama, menjadi 3 buah kelas, antara lain: penggunaan masker penuh (hijau), penggunaan masker tidak penuh (kuning), dan tanpa menggunakan masker (merah).

Tahapan klasifikasi ini menggunakan arsitekur Convolutional Neural Network yang terdiri dari:
  1. Konvolusi ; jumlah kernel : 128, ukuran kernel : 3 x 3
  2. MaxPooling ; stride : 2
  3. Konvolusi ; jumlah kernel : 64, ukuran kernel : 3 x 3
  4. MaxPooling ; stride : 2
  5. Konvolusi ; jumlah kernel : 32, ukuran kernel : 3 x 3
  6. MaxPooling ; stride : 2
  7. Konvolusi ; jumlah kernel : 16, ukuran kernel : 3 x 3
  8. MaxPooling ; stride : 2
  9. Konvolusi ; jumlah kernel : 8, ukuran kernel : 3 x 3
  10. FC layer ; 8 neuron
Tatacara Deteksi
  1. Arahkan wajah Anda pada kamera dan posisikan badan tegak sejauh 2 meter
  2. Tunggu dalam beberapa menit, sistem akan mendeteksi penggunaan masker Anda
